# Linktree-style Landing Page

A simple Linktree-style landing page for sharing social and website links.

## Project Structure

- `index.html` — main page markup
- `style.css` — page styling
- `assets/profile.jpg` — profile image used on the page
- `assets/icons/` — SVG icons for TikTok, Instagram, Facebook, LinkedIn, and Website

## Features

- Clean, centered layout
- Custom link buttons with brand icons
- Responsive design for mobile and desktop

## Customization

To update the links or labels, edit `index.html` inside the `<nav class="links">` section.

To change the profile image, replace `assets/profile.jpg`.

To update the styling, edit `style.css`.

## Usage

Open `index.html` in your browser or serve the folder from a local web server.

## Notes

- The project uses inline SVG icons stored in `assets/icons/` for consistent icon display.
- The current links point to TikTok, Instagram, Facebook, LinkedIn, and Website.
